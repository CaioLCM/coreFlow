# ABOUT
Backend system built with NestJS + PrismaORM, designed as a scalable foundation for corporate systems with authentication and role-based access control.

## Purpose
This project aims to build a meaningful and well-structured backend system while using AI as a technical guide to simulate real-world engineering decision-making and learning processes.

## Stack
- Node.js
- NestJs
- Prisma ORM
- PostgreSQL
- Docker

## Running the project
- docker-compose up db
- npm install
- npm run start:dev
